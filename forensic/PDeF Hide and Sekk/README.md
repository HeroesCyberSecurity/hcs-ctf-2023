## Solution

Kita diberikan sebuah file pdf, namun pada saat melakukan `CTRL+A` di pdf akan terlihat sebuah keanehan

![POC 1](images/POC%201.jpg)

Kita dapat mengcopy, kemudian paste pada notepad
